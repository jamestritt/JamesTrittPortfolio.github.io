About

Using the Cincinnati Police PDI (Police Data Initiative) Crime Incidents database, this data represents reported Crime Incidents in the City of Cincinnati daily. Using this data, the my site allows users to filter for useful data that user would want to know when searching for a neighborhood. This website use JavaScript for programming, handlebars to compile into templates from javascript functions and HTML & CSS for basic website formatting. The concept of this website is to sort, organize and display meaningful information when searching for a particular neighborhood in Cincinnati.

Using data to help people see what kind of crime is being commited in which neighborhood. From this, new residents of cincinnati can base where they would like to live at.
